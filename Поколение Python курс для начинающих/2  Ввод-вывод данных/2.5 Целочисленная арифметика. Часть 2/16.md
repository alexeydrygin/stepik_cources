## Дано пятизначное число, которое хранится в переменной n. 

В переменных a, b, c, d, e хранятся:

a – число десятков тысяч (первая цифра),
b – число тысяч (вторая цифра),
c – число сотен (третья цифра),
d – число десятков (четвертая цифра),
e – число единиц (пятая цифра).
Установите соответствие между элементами первого и второго столбцов.


> Обратите внимание: если n – пятизначное число, то первая цифра числа равна n % 100000 // 10000, что совпадает с n // 10000.

a = n // 10000
b = n % 10000 // 1000
c = n % 1000 // 100
d = n % 100 // 10
e = n % 10
